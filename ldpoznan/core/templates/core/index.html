{% blocktrans %}Wait, what's a game jam?{% endblocktrans %}

{% blocktrans %}The idea is simple — programmers, designers, artists meet together to make a game in a short time. Ludum Dare is divided into two categories:

  • 48h compo — 2 days, you have to work on your game by yourself, all content has to be created during the duration of the event
  • 72h jam — 3 days, teams of any size allowed, you can use freely available content like sprites
